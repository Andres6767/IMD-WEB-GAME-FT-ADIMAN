Flappy Bird Remake

Game Instructions & Description

Use **X**, or **ARROW UP** to make the bird fly up. **Avoid the pipes!**

  • Score increases by 1 for each pipe successfully passed.
  • Game ends if the bird hits a pipe or the ground.
  • Press **X**,or **ARROW UP** to restart after game over.
